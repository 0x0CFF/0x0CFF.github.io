GNU/Linux

从 Rust 要进入 Linux 6.1 内核开始关注，到现在学习 Rust 第 10 天，把基本的概念都简单地过了一遍，发现这个语言许多方面的设计令人一眼钟情

  • 拥有一个强大的项目管理工具 Cargo ，一套 Cargo 命令完成：创建项目（同时会很贴心地创建一个 Git 仓库）、包管理、检查代码、单元测试、编译项目
  • 拥有一个严格且贴心的编译器，会告诉开发者哪里的代码出错了、为什么错了、应该怎么改
  • 代码的语法风格符合我个人的口味
  • 错误处理设计的非常简洁优雅
  • 非面向对象，却能够用结构体（Struct）、特性（Trait）、方法（Impl）实现面向对象，并且更加灵活

当然也存在一些我个人非常不习惯的地方

  • 字符串不作为一种类型存在，不好理解
  • Lambda 表达式的写法不怎么优雅（对比 Python）
  • 所有权和生命周期作为 Rust 的核心内容，非常的难学
